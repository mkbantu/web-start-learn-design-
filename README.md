# WebStart Learn Design

WebStart Learn Design is an e-learning platform for beginners who want to learn HTML, CSS, and JavaScript in a simple and practical way. It is built as a course-style website with learning pages, lesson sections, navigation, and a registration form for students who want to follow the course.

## Why I built it

This project is a university assignment. I chose to build an e-learning platform for University of Rwanda students so they can have a clear, beginner-friendly place to learn web design basics and review the core topics used to build websites.

## What the project does

- Introduces web design basics in one place.
- Provides dedicated pages for HTML, CSS, and JavaScript.
- Includes a home page that guides users through the learning path.
- Includes a student registration form for course sign-up.
- Uses a simple folder structure to keep HTML, CSS, and images organized.

## Technology used

- HTML
- CSS
- YouTube embed for the hero video on the home page
