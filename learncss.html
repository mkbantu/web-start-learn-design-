CSS Introduction

CSS (Cascading Style Sheets) describes how HTML elements are displayed on screen. It separates content from design.

/* CSS saved as styles.css */
body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}
h1 { color: navy; }
p  { color: grey; font-size: 16px; }
  • CSS stands for Cascading Style Sheets
  • CSS saves a lot of work — one file can style many pages
  • "Cascading" means styles flow from parent to child elements

CSS Syntax

A CSS rule has a selector and a declaration block containing property/value pairs.

selector {
  property: value;
  property: value;
}

h1         { color: blue; }      /* element */
.highlight { color: red; }       /* class */
#title     { font-size: 2rem; }  /* id */
*          { box-sizing: border-box; } /* universal */

CSS Selectors

p          { }   /* all <p> elements */
.card      { }   /* class="card" */
#hero      { }   /* id="hero" */
*          { }   /* all elements */
h1, h2, p  { }   /* grouping — applies to all three */
  • Element selectors are least specific
  • Class selectors (.name) are reusable across many elements
  • ID selectors (#name) target one unique element
  • Grouped selectors share the same rules — saves repetition

CSS How To

<!-- External (recommended) -->
<link rel="stylesheet" href="styles.css">

<!-- Internal -->
<style>
  p { color: red; }
</style>

<!-- Inline -->
<p style="color:red;">Red text</p>
  • External: one file styles the whole site — best practice
  • Internal: styles one page only — placed in <head>
  • Inline: highest priority — hardest to maintain
  • Cascade order: inline > internal/external (last wins) > browser default

CSS Comments

/* Single-line comment */

/*
  Multi-line comment
  spanning several lines
*/

p { color: red; } /* inline comment */
  • Comments are ignored by the browser
  • Use them to explain complex rules or separate sections

CSS Backgrounds

div {
  background-color: #f0f0f0;
  background-image: url('bg.jpg');
  background-repeat: no-repeat;   /* repeat | repeat-x | repeat-y */
  background-position: center top;
  background-size: cover;         /* contain | auto | px/% */
  background-attachment: fixed;   /* scroll | local */
}

/* Shorthand */
background: #f0f0f0 url('bg.jpg') no-repeat center/cover fixed;
  • cover — scales to fill, may crop
  • contain — fits inside, may leave gaps
  • fixed — background stays fixed on scroll (parallax effect)

CSS Borders

border: 2px solid #333;          /* shorthand */
border-width: 2px;
border-style: solid;             /* dotted | dashed | double | groove | none */
border-color: red;
border-top: 3px dashed blue;     /* individual sides */
border-radius: 8px;              /* rounded corners */
border-radius: 50%;              /* circle */

CSS Margins

Margin is the space outside an element's border.

margin: 20px;                    /* all sides */
margin: 10px 20px;               /* top/bottom  left/right */
margin: 5px 10px 15px 20px;      /* top right bottom left */
margin-top: 10px;
margin: 0 auto;                  /* center block element horizontally */
margin: -10px;                   /* negative margin allowed */
  • Vertical margins between block elements collapse to the larger value
  • margin: 0 auto centers a block element (needs a set width)

CSS Padding

Padding is the space inside an element between content and border.

padding: 20px;
padding: 10px 20px;
padding: 5px 10px 15px 20px;     /* top right bottom left */
padding-left: 16px;
  • Padding increases the element's size (unless box-sizing: border-box)
  • Negative padding is not allowed

CSS Height / Width

div {
  width: 300px;
  height: 200px;
  max-width: 100%;     /* won't overflow parent */
  min-height: 50px;
  width: 50%;          /* relative to parent */
  width: 100vw;        /* viewport width */
  height: 100vh;       /* viewport height */
}
  • Default: width: auto (stretches to fill parent for block elements)
  • Use max-width: 100% on images for responsive behavior

CSS Box Model

Every element is a rectangular box: content → padding → border → margin.

/* Default: width = content only */
div { width: 300px; padding: 20px; border: 5px solid; }
/* Total visual width = 300 + 40 + 10 = 350px */

/* Border-box: width includes padding + border */
* { box-sizing: border-box; }
div { width: 300px; padding: 20px; border: 5px solid; }
/* Total visual width = 300px exactly */
  • Always add * { box-sizing: border-box; } to your CSS reset

CSS Outline

Outline is drawn outside the border. It does not affect layout (takes no space).

outline: 3px solid red;
outline-width: 3px;
outline-style: dotted;
outline-color: red;
outline-offset: 5px;   /* gap between border and outline */
outline: none;         /* removes focus outline (use carefully!) */
  • Unlike borders, outlines don't change element size
  • Commonly seen on focused form inputs — don't remove without a replacement

CSS Fonts

font-family: 'Segoe UI', Arial, sans-serif;
font-size: 16px;        /* px | em | rem | % */
font-weight: bold;      /* normal | 100-900 */
font-style: italic;     /* normal | oblique */
font-variant: small-caps;

/* Shorthand: style variant weight size/line-height family */
font: italic bold 16px/1.5 Arial, sans-serif;

/* Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Roboto');
font-family: 'Roboto', sans-serif;
  • Always end font stack with a generic family: serif, sans-serif, monospace
  • Use rem for font sizes — scales with user browser settings

CSS Display

display: block;          /* full width, new line */
display: inline;         /* content width, same line, no w/h */
display: inline-block;   /* content width, same line, has w/h */
display: none;           /* hidden, no space taken */
display: flex;           /* flexbox container */
display: grid;           /* grid container */
display: table;          /* behaves like <table> */
  • visibility: hidden hides but keeps space; display: none removes it completely

CSS Position

position: static;    /* default — normal flow */
position: relative;  /* offset from its normal position */
position: absolute;  /* relative to nearest positioned ancestor */
position: fixed;     /* relative to viewport, stays on scroll */
position: sticky;    /* relative until scroll threshold, then fixed */

/* Use with: top, right, bottom, left */
.box {
  position: absolute;
  top: 20px;
  left: 50px;
}
  • relative keeps its space in the flow
  • absolute is removed from the flow
  • fixed is great for headers and back-to-top buttons
  • sticky requires a top value to activate

CSS Z-index

Z-index controls the stacking order of positioned elements. Higher = on top.

.modal   { position: fixed; z-index: 1000; }
.overlay { position: fixed; z-index: 999; }
.header  { position: sticky; z-index: 100; }
  • Only works on positioned elements (relative, absolute, fixed, sticky)
  • Default z-index is auto (= 0)
  • Negative z-index places an element behind others

CSS Overflow

overflow: visible;  /* default — content spills out */
overflow: hidden;   /* clips overflowing content */
overflow: scroll;   /* always shows scrollbars */
overflow: auto;     /* scrollbars only when needed */

overflow-x: hidden; /* clip horizontal only */
overflow-y: auto;   /* scroll vertical only */

CSS Float

img { float: left; margin: 0 16px 8px 0; }

/* Clearfix — parent collapses when all children float */
.clearfix::after {
  content: "";
  display: table;
  clear: both;
}
  • Float is mainly used to wrap text around images today
  • For layouts, prefer Flexbox or Grid instead
  • Always clear floats to prevent parent collapse

CSS Inline-block

inline-block sits on the same line as siblings but respects width and height.

.nav-item {
  display: inline-block;
  width: 100px;
  height: 40px;
  text-align: center;
  line-height: 40px;
}

CSS Align

/* Center a block element */
div { width: 300px; margin: 0 auto; }

/* Center text */
p { text-align: center; }

/* Center with Flexbox */
.container {
  display: flex;
  justify-content: center;  /* horizontal */
  align-items: center;      /* vertical */
  height: 100vh;
}

/* Center with Grid */
.container {
  display: grid;
  place-items: center;
}

CSS Flexbox

Flexbox is a one-dimensional layout system for arranging items in a row or column.

/* Container */
.container {
  display: flex;
  flex-direction: row;          /* column | row-reverse | column-reverse */
  justify-content: space-between; /* flex-start | center | space-around */
  align-items: center;           /* flex-start | flex-end | stretch */
  flex-wrap: wrap;               /* nowrap | wrap-reverse */
  gap: 16px;
}

/* Items */
.item {
  flex: 1;            /* grow and shrink equally */
  flex: 0 0 200px;    /* don't grow, don't shrink, 200px */
  align-self: flex-end;
  order: 2;
}

CSS Grid

Grid is a two-dimensional layout system for rows and columns simultaneously.

/* Container */
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;        /* 3 columns */
  grid-template-columns: repeat(3, 1fr);     /* same */
  grid-template-rows: 100px auto 60px;
  gap: 16px;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

/* Items */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }

/* Span cells */
.wide { grid-column: 1 / 3; }  /* span 2 columns */
.tall { grid-row: 1 / 3; }     /* span 2 rows */

CSS Combinators

div p     { }  /* descendant — any <p> inside <div> */
div > p   { }  /* child — only direct <p> children */
h2 + p    { }  /* adjacent sibling — <p> immediately after <h2> */
h2 ~ p    { }  /* general sibling — all <p> after <h2> */

CSS Attribute Selectors

[type]           { }   /* has type attribute */
[type="text"]    { }   /* type equals "text" */
[class~="card"]  { }   /* class contains word "card" */
[href^="https"]  { }   /* href starts with "https" */
[href$=".pdf"]   { }   /* href ends with ".pdf" */
[href*="google"] { }   /* href contains "google" */

CSS Specificity

When rules conflict, the more specific rule wins.

/* Specificity score (higher = wins) */
*            { }   /* 0 */
p            { }   /* 1 */
.class       { }   /* 10 */
#id          { }   /* 100 */
style=""         /* 1000 */
!important       /* overrides all */

/* Example: which color wins? */
p           { color: black; }   /* 1 */
.text       { color: blue; }    /* 10 ← wins */
#main p     { color: red; }     /* 101 ← wins over .text */
  • Avoid !important — it breaks the natural cascade
  • When specificity is equal, the last rule in the file wins

CSS Transitions

Transitions animate property changes smoothly.

/* transition: property duration timing-function delay */
button {
  background: blue;
  transition: background 0.3s ease;
}
button:hover { background: red; }

/* Multiple transitions */
div {
  transition: width 0.5s ease, opacity 0.3s linear;
}

/* All properties */
div { transition: all 0.3s ease-in-out; }
  • Timing functions: ease, linear, ease-in, ease-out, ease-in-out

CSS Animations

/* Define keyframes */
@keyframes slideIn {
  from { transform: translateX(-100%); opacity: 0; }
  to   { transform: translateX(0);     opacity: 1; }
}

@keyframes pulse {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.1); }
  100% { transform: scale(1); }
}

/* Apply animation */
.box {
  animation: slideIn 0.5s ease forwards;
  animation: pulse 2s ease-in-out infinite;
  /* name duration timing fill-mode iteration-count */
}

CSS Variables

/* Define on :root for global scope */
:root {
  --primary: #0f3460;
  --accent:  #e94560;
  --spacing: 16px;
  --radius:  8px;
}

/* Use with var() */
button {
  background: var(--primary);
  padding: var(--spacing);
  border-radius: var(--radius);
}

/* Fallback value */
color: var(--text-color, black);

CSS Responsive / Media Queries

/* Mobile first: base styles, then override for larger screens */
.container { width: 100%; }

@media (min-width: 768px) {
  .container { width: 750px; }
}
@media (min-width: 1024px) {
  .container { width: 960px; }
}

/* Dark mode */
@media (prefers-color-scheme: dark) {
  body { background: #1a1a2e; color: #fff; }
}

/* Print */
@media print {
  nav, footer { display: none; }
}

CSS Math Functions

width: calc(100% - 60px);          /* arithmetic */
width: min(500px, 100%);           /* smaller of two */
width: max(300px, 50%);            /* larger of two */
font-size: clamp(1rem, 2.5vw, 2rem); /* min, ideal, max */
  • calc() — mix units (px, %, em)
  • clamp() — fluid typography / sizing
